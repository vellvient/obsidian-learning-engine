# Resources (A-Level / admission-test focus)

The resource list behind the reference build. All public sites; availability and
URLs drift — treat this as a map, not gospel.

## Past papers & mark schemes ("the big three")

- **Save My Exams** — savemyexams.com/a-level/ (topic questions + model answers)
- **PapaCambridge** — pastpapers.papacambridge.com (CAIE: QP, MS, examiner reports,
  grade thresholds, all years)
- **Physics & Maths Tutor (PMT)** — physicsandmathstutor.com (topic compilations;
  also strong admission-test sections)

Fallbacks: pastpapers.co · GCE Guide · Crack A Level.

## Admission tests (TMUA/MAT/STEP pattern)

- Official prep hub first — e.g. esat-tmua.ac.uk/tmua-preparation-materials/
  (official past papers, computer-based-test specimen, the "Notes on Logic and
  Proof" / "Notes on Mathematics" documents that make excellent Phase-1 sources)
- PMT admissions sections (video solutions) · exams.ninja · university support
  pages (e.g. Warwick's TMUA walkthroughs) · tmua.co.uk (topic-level feedback)

## Recall & revision tools (alternatives/complements)

Anki (the FSRS reference implementation lives there) · Seneca · ZNotes.

## Strategy notes from the reference build

- Time split that worked: **~70% past papers / 20% topical drilling on weak spots /
  10% notes + flashcards.** The graph + question bank exist to make the 20% surgical.
- **Examiner reports** are the highest-density mistake-analysis material an exam
  board publishes — feed recurring examiner complaints into the error log as
  pre-emptive weak spots.
- Distinctive exam components (e.g. a logic/proof paper) need dedicated weekly time
  even when the main subject feels strong — a strong-subject halo doesn't transfer
  to format-specific skills.
